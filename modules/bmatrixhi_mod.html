Description

MODULE BmatrixHI_mod (prefix=’bhi’ category=’2. B and R matrices’)

Purpose

Performs transformation from control vector to analysis increment using the background-error covariance matrix based on homogeneous and isotropic correlations. This is the Global version. A separate module exists for limited-area applications.
